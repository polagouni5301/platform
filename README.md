# HireIq Separated Frontends

This project has been separated into two independent frontend applications:

- **frontend_candidate**: The portal for job seekers. Runs on port 8081.
- **frontend_recruiter**: The portal for recruiters and the main landing page. Runs on port 8080.

## Getting Started

To run either application, navigate into its directory and use the standard commands:

```bash
cd frontend_candidate
bun install
npm run dev
```

or

```bash
cd frontend_recruiter
bun install
npm run dev
```
